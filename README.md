# Blue_Block_Solver
This is a program that automatically solves N puzzles in Blue Block game (game link: https://apps.apple.com/us/app/blue-block/id320111844). It uses a BFS algorithm. This is done in C++.

**Note:** It is recommended to debug line by line and enable permissions for using mouse manipulations. The pixels can vary from different computer, especially how the program captures the screen and detects blocks.

To compile, you type "make". To clean, you type "make clean".
